
Barbearia do Padre - site pronto para upload na HostGator
Conteúdo do ZIP:
- index.html
- style.css
- script.js
- .htaccess
- assets/ (logo.png, img1.jpg ... img24.jpg, video_poster.jpg)
- README.txt (este arquivo)

Instruções de upload:
1) No cPanel da HostGator, abra File Manager -> public_html
2) Faça upload do arquivo 'barbeariadopadre_site.zip' (arquivo ZIP que você baixou)
3) Após o upload, selecione o ZIP e clique em 'Extract' para descompactar no public_html
4) Verifique se existe public_html/index.html
5) Teste no navegador: http://barbeariadopadre.top
6) Ative SSL: cPanel -> SSL/TLS Status -> Run AutoSSL
7) Substitua os arquivos de mídia:
   - assets/tour.mp4 (adicione aqui seu vídeo de tour real)
   - assets/logo.png (substitua pela sua logo final com fundo transparente)
   - assets/img*.jpg (substitua pelas fotos reais)
8) Se quiser forçar HTTPS, o .htaccess já está configurado.
